##  What is the difference between var, let, and const?
### Answer:var, let, and const are used to create variables. var is the old way. let can be changed later, but const cannot be changed after it is set.
##  What is the spread operator (.)?
### Answer:The spread operator is used to copy or combine values from an array or object easily.
##  What is the difference between map(), filter(), and forEach()?
### Answer:map() changes items and returns a new array.
filter() returns items that match a condition.
forEach() just loops through the array.
##  What is an arrow function?
### Answer:An arrow function is a shorter way to write a function in JavaScript.
##  What are template literals?
### Answer:Template literals are a way to write strings where you can easily add variables inside the text.
